
# Insurance Claim Prediction Model

## Project Summary (PDF)
[View Project Summary](./insuranceclaimprojectsummary%20(1).pdf)

## Description
This project develops a logistic regression model to estimate insurance claim probability using policyholder demographics, driving behavior, and vehicle characteristics.

## Key Features
- Data preprocessing and handling of missing values  
- Exploratory data analysis of key risk factors  
- Logistic regression (GLM with logit link)  
- Model evaluation using AUC, AIC, lift, and calibration  

## Key Findings
- Drivers with more experience had significantly lower claim probability  
- Male drivers had higher claim odds than female drivers  
- Higher mileage and more speeding violations increased risk  
- Newer vehicles were associated with lower claim probability  

## Model Evaluation
- AUC used to assess discrimination ability  
- Decile lift chart used to evaluate ranking performance  
- Calibration plot used to compare predicted vs actual probabilities
